# 07 - 列表渲染

Vue.js 列表渲染详解

## 学习内容

1. **v-for 基础** - 基于数组渲染列表
2. **对象数组** - 渲染对象数组
3. **遍历对象** - 遍历对象属性
4. **范围值** - 基于数字范围渲染
5. **template 上的 v-for** - 渲染多个元素
6. **数组变化侦测** - 响应式数组方法
7. **嵌套 v-for** - 多层嵌套列表
8. **key 的重要性** - 为什么需要 key

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
