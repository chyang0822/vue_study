# 09 - 表单输入绑定

Vue.js 表单输入绑定详解

## 学习内容

1. **v-model 基础** - 双向数据绑定
2. **文本输入** - input 和 textarea
3. **复选框** - 单个和多个复选框
4. **单选按钮** - radio
5. **选择器** - select 单选和多选
6. **修饰符** - .lazy、.number、.trim
7. **实际应用** - 完整表单示例

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
