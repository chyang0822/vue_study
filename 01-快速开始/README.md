# 01 - 快速开始

Vue.js 快速开始教程示例

## 学习内容

1. **声明式渲染** - 使用模板语法将数据渲染到 DOM
2. **响应式基础** - Vue 自动跟踪状态变化并更新 DOM
3. **Attribute 绑定** - 使用 `v-bind` 或 `:` 绑定动态属性
4. **事件监听** - 使用 `v-on` 或 `@` 监听 DOM 事件

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
