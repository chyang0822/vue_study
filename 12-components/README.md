# 12 - 组件基础

Vue.js 组件基础详解

## 学习内容

1. **定义组件** - 创建可复用组件
2. **使用组件** - 注册和使用
3. **Props** - 父传子通信
4. **事件** - 子传父通信
5. **插槽** - 内容分发
6. **动态组件** - component :is
7. **实际应用** - 完整组件示例

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
