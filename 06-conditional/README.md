# 06 - 条件渲染

Vue.js 条件渲染详解

## 学习内容

1. **v-if** - 条件性渲染
2. **v-else** - else 区块
3. **v-else-if** - else if 区块
4. **template 上的 v-if** - 包装多个元素
5. **v-show** - 基于 CSS 的显示切换
6. **v-if vs v-show** - 两种方式的对比
7. **实际应用** - 登录状态示例

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
