# 11 - 侦听器

Vue.js 侦听器详解

## 学习内容

1. **watch 基础** - 侦听响应式数据
2. **深层侦听** - deep 选项
3. **即时回调** - immediate 选项
4. **watchEffect** - 自动追踪依赖
5. **停止侦听器** - 手动停止
6. **侦听多个源** - 同时侦听多个数据
7. **实际应用** - 搜索、数据同步等

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
