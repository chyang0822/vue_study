# 03 - 响应式基础

Vue.js 响应式系统详解

## 学习内容

1. **ref()** - 创建响应式引用
2. **reactive()** - 创建响应式对象
3. **深层响应性** - 嵌套对象的响应式
4. **ref vs reactive** - 两种方式的对比
5. **toRefs()** - 解构响应式对象
6. **toRef()** - 单个属性转 ref
7. **响应式判断** - isRef、isReactive

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
