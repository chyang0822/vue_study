# 08 - 事件处理

Vue.js 事件处理详解

## 学习内容

1. **监听事件** - v-on / @ 指令
2. **内联事件处理器** - 直接编写 JavaScript
3. **方法事件处理器** - 指向组件方法
4. **访问事件对象** - $event 变量
5. **事件修饰符** - .stop、.prevent、.once 等
6. **按键修饰符** - .enter、.tab、.esc 等
7. **系统按键修饰符** - .ctrl、.alt、.shift、.meta
8. **鼠标按键修饰符** - .left、.right、.middle
9. **.exact 修饰符** - 精确的系统按键组合

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
