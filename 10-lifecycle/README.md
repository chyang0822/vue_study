# 10 - 生命周期

Vue.js 生命周期钩子详解

## 学习内容

1. **生命周期图示** - 组件生命周期流程
2. **onMounted** - 组件挂载后
3. **onUpdated** - 组件更新后
4. **onUnmounted** - 组件卸载前
5. **onBeforeMount** - 挂载前
6. **onBeforeUpdate** - 更新前
7. **onBeforeUnmount** - 卸载前
8. **实际应用** - 定时器、API 调用等

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
