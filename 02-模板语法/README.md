# 02 - 模板语法

Vue.js 模板语法详解

## 学习内容

1. **文本插值** - 使用 `{{ }}` 双大括号语法
2. **原始 HTML** - 使用 `v-html` 指令
3. **Attribute 绑定** - 使用 `v-bind` 或 `:` 简写
4. **JavaScript 表达式** - 在模板中使用表达式
5. **指令** - 带有 `v-` 前缀的特殊 attribute
6. **动态参数** - 使用方括号指定动态参数

## 项目设置

```sh
yarn install
```

### 开发模式运行

```sh
yarn dev
```

### 构建生产版本

```sh
yarn build
```
